# To Do List Web Application

HTML | Bootstrap5 | Python | Flask | SQLAlchemy | SQLite | CRUD |

# Instrução
- Acesse o diretório do projeto através do Terminal/VSCODE.
- Para criar o ambiente virtual (windows), digite: python -m venv venv
- Para ativar o ambiente virtual (windows), digite: .\venv\Scripts\Activate.ps1
- Instale a biblioteca Flask, digite: pip install flask
- Instale a biblioteca Flask-SQLAlchemy: pip install flask-sqlalchemy==2.5.1
- Digite (ambiente virtual = ambiente de desenvolvimento): set FLASK_ENV=development

- Criar a base de dados dentro do diretório
- digite no terminal: python
- digite no terminal: from app import db
- digite no terminal: from app import db
- digite no terminal: db.create_all()
- Aparecerá o arquivo sql.db na pasta do projeto
- digite exit()

- Para rodar o projeto, digite: flask run
